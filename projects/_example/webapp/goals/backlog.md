# Goal backlog — webapp (example)

Run each as: `cnc goal "<goal>" --project <org>/<repo>` (or `/goal <goal>` in a commander session).
Every goal implies: use parallel agents, merge each PR sequentially with `claudetm merge-pr`, deep full work — 100% complete, no half-done features.

1. Take each tab in the settings feature and fix it end to end; add missing tests.
2. Find logic bugs all around and fix them.
3. Finish the dashboards — find bugs, refactor, 100% complete.
4. CSS craft: rem everywhere, functions/mixins/variables, SOLID/SRP CSS, reusable; better UX — shadows, transition durations, animations.
5. Add missing unit/integration tests across the app.
